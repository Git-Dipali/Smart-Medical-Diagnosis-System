# Smart-Medical-Diagnosis-System
Smart Medical Diagnosis System is an AI-based Python application that predicts diseases from user symptoms using Bayesian Networks and Knowledge Representation. The system performs probability-based diagnosis through a terminal interface and provides intelligent, fast, and structured medical decision support.
The Smart Medical Diagnosis System is an AI-based healthcare application developed using Python. The system is designed to predict possible diseases based on symptoms entered by the user through a terminal-based interface. The main aim of the project is to demonstrate the use of artificial intelligence techniques such as Bayesian Networks and Knowledge Representation in the field of medical diagnosis.

The system contains a medical knowledge base that stores information about various diseases and their related symptoms. When the user enters symptoms, the system analyzes them and calculates the probability of different diseases using Bayesian probability concepts. The disease with the highest probability is displayed as the most likely diagnosis along with other possible diseases and their probability values.

Knowledge Representation is used to organize diseases, symptoms, and probabilities in a structured format, allowing the system to perform intelligent reasoning. Bayesian Networks help the system manage uncertain and overlapping symptoms effectively, making the diagnosis process more accurate and logical.

The project includes features such as symptom-based disease prediction, probability-based results, a menu-driven terminal interface, and an expandable knowledge base where new diseases and symptoms can be added easily. The system is lightweight, user-friendly, and modular in design.

This project demonstrates how artificial intelligence can be applied in healthcare for preliminary diagnosis and decision support. Although the system is developed for educational purposes and does not replace professional medical consultation, it effectively showcases the practical implementation of probabilistic reasoning and knowledge-based systems in medical applications.
